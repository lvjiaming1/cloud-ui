### 基本用法

``` html
<u-divider title="内容"></u-divider>
```

### 左

``` html
<u-divider dashed="b" content-position="left" title="左">
</u-divider>
```

### 竖向

``` html
<u-divider dashed="b" content-position="left" title="左" direction="column">
</u-divider>
```

### 竖向

``` html
<u-divider content-position="center" title="中" direction="column">
</u-divider>
```
